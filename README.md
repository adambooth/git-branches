# git-branches

overview of git and branches

1. create a repo

2. copy the ssh key

3. git clone SSHKEY ie. git@github.com:adambooth/git-branches.git

// This links both local and git projects ready to push updates

4. git add .

5. git commit -m "update description here"

6. git push

// This adds new updates to git

7. git status

// This shows how far infront/behind your local project and git are of each other

8. git restore

// this restore it to the previous commit in case you made a bad push
